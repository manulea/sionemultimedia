Sione Manulea's Multimedia portfolio!

Welcome to my webpage! I created this website using bootstrap and css

The navbar is the typical navbar item list from the bootpage website which you can view here

And the rest of the website, I coded myself with the help of Google and previous website making experience

I dedicated myself to using bootstrap because it seemed to be one of the better methods where complete customizbility was within my control meaning, I can shape my website in a way where I can decide what things do what, and where things go where

Jekyll came to mind when making a website, because of it's ease of user creation. But with this website, it wasn't needed to have such functionality, as I decided this portfolio would be a place to showcase my multimedia work for all the world to see, meaning I would not need to restrict or filter users/viewers. This is why I preferred to code purely in bootstrap / html / css!

I decided having a navigation menu at the top would be more fulfilling because it seemed like a more productive way for the user to navigate around the website efficiently

On this website, there are five main categories which are the basis of my Multimedia work

These categories are, lab designs, project designs, assignments, and assessments
